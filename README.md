# 🧠 Lyneitor - Agente IA Fullstack Élite

Este proyecto es una aplicación web responsiva e inteligente que funciona como un Agente de Inteligencia Artificial experto en desarrollo de software, diseño web, bases de datos (SQL) y creación de arquitecturas de IA.

## 🚀 Tecnologías utilizadas
* **Frontend:** HTML5, Tailwind CSS, JavaScript Moderno.
* **Backend (Serverless):** Node.js corriendo en Vercel API Functions.
* **Cerebro Artificial:** Google GenAI SDK (Modelos Gemini).

## 📁 Estructura del repositorio
* `/public/index.html`: Interfaz gráfica con sistema de copiado de código dinámico integrado.
* `/api/chat.js`: Lógica del servidor que conecta de manera segura con las credenciales de la API de inteligencia artificial.
* `package.json`: Archivo de configuración del entorno y control de dependencias.

## 🛠️ Instalación y despliegue rápido
1. Sube estos archivos a tu repositorio personal de GitHub.
2. Ve a **Vercel.com**, vincula tu cuenta e importa este repositorio.
3. Configura la variable de entorno (`Environment Variables`) con el nombre `GEMINI_API_KEY` y pega allí tu llave privada de Google AI Studio.
4. ¡Haz clic en **Deploy** y disfruta de Lyneitor en la nube de forma totalmente gratuita!
